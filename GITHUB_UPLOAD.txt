❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ:

Q: Что если GitHub запросит пароль?
A: Введите пароль от GitHub (или используйте GitHub CLI token)

Q: Ошибка "Permission denied"?
A: Используйте GitHub CLI: gh auth login

Q: Ошибка "refusing to merge unrelated histories"?
A: Выполните:
   git pull origin main --allow-unrelated-histories
   git push -u origin main

Q: Что такое "origin" и "main"?
A: 
   - origin = имя удаленного репо на GitHub
   - main = имя основной ветки

Q: Могу ли я потом изменить?
A: Да! После загрузки можете редактировать файлы прямо на GitHub.
   Все изменения можно вернуть обратно локально через git pull.
